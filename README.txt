steps to use Battery health monitor:

1. install python version > 3.8.0
2. open CMD and change directory to batteryFull_notifier folder (i.e cd  C:\Users\admin\AppData\Local\Programs\Python\Python38\practice programs\batteryFull_notifier)
3. install dependecies using "pip install -r requirements.txt"
4. Run the "Battery_Health_Monitor.pyw" file by double click.
5. Once the application window opens click on toggle button which moves to on and starts battery monitoring.
6. Once battery reaches 95% and still the adaptor plugged in we will get notified every 1 min until you remove the adaptor.
7. if adaptor is not plugged in and battery reaches below 30% we will get notification.
